# Gif-Gif

### Gifs search application.

You can search for any gifs, register & login to add them to **favorites**!

Tech used: 

* React
* Node.js
* Express

Worth mentioning:

* JWT (jsonwebtoken)
* React Testing Library
* React Hook Form

### Gifs are fetched from the Giphy API 
https://developers.giphy.com/
